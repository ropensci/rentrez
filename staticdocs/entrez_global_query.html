See how many hits there are for a given term across all NCBI Entrez databases

Usage

entrez_global_query(term)

Arguments

term
the search term to use

Value

a named vector with counts for each a database

Description

Constructs a url with the given arguments, and downloads xml record returned by that url. See the package-level documentation for general advice on using the EUtils functions.

Examples

mini_elephants <- entrez_global_query(term="Dwarf Elephant")
